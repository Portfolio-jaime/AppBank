¿Qué formatos de descarga necesitas?

Puedes seleccionar varios

A

PDF — Resumen ejecutivo

Página con nombre del cliente, parámetros del crédito, KPIs principales y tabla de amortización completa. Listo para imprimir o enviar.

B

Excel / CSV — Datos crudos

Hoja de cálculo con cada cuota, interés, abono y saldo mes a mes. Ideal para análisis adicional o importar a otro sistema.

C

Compartir — Link o JSON

Exporta los parámetros como URL (con query params) o JSON para guardar y retomar la simulación exacta más tarde.
